# Causal steering experiment (v2 stretch)

Steers the L10 residual stream so that the trained `within_task_fraction` probe's readout at every token equals a target value, then re-runs Gemma-9B-IT on the v1 test traces. Three conditions: `no_steer` (baseline), `early_steer` (target=0.1, 'pretend you just started this case'), `late_steer` (target=0.9, 'pretend you're about to finish this case'). All other context unchanged.

## Interpretation

If `early_steer` lowers the calibration gap relative to `no_steer`, AND `late_steer` raises it symmetrically, then the within_task_fraction direction is causally implicated in the fatigue calibration gap. If only one direction has an effect, or both perturb in the same direction, the story is messier — but still informative about how the model uses this representation.
